
# Budget & Expense Management System

A console-based C++ system to track income, expenses, and savings with detailed analytics.

## Features
- Add, update, and remove income and expense entries
- Categorized expense tracking (Food, Travel, Bills, etc.)
- Overspending alerts and savings analytics
- Monthly and yearly summaries
- Menu-driven CLI interface

## Technologies Used
- C++ Classes, Objects, Enums, Vectors
- Functions and Modular Structure

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o BudgetSystem`
3. Run: `./BudgetSystem`
